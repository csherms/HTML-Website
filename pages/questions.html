HTML Terminology and Definitions

  1. What do HTML and CSS stand for?
    • HTMl Stands for Hypertext Markup Language. And CSS Stnads for Cascading Style Sheets.

  2. Would you rather use HTML or CSS, for putting paragraphs of text on a webpage?
    • I would rather use HTML because creating the basic information of the page (such as text) is the main purpose of HTML.

  3. Would you rather use HTML or CSS, to change the font and background color of a button?
    • I would rather use CSS as it is the most efficent way to style the various elements of an HTML page.

  4. What is an HTML tag?
    • An HTML tag is the markup used to indicate the opening and closing of an HTML element.

  5. What are the three parts of an HTML element?
    • The opening tag, the content, and the closing tag.

  6. What is the purpose of the doctype declaration?
    • It tells the browser what version of HTML the document is written in.

  7. What is the HTML element?
    • An HTML element is defined by a start tag, some content, and an end tag.

  8. What is the purpose of the head element?
    • It describes properties of the document such as the title, the document toolbar, and additional meta-information.

  9. What is the purpose of the body element?
    • It contains all the contents of an HTML document, such as headings, paragraphs, images, etc.

  10. How do you create a paragraph in HTML?
    • By using the paragraph start tag, adding in the content and closing the paragraph with the paragraph's closing tag.

  11. How do you create a heading in HTML?
    • Headings follow the standard opening tag followed by content and then the closing tag syntax, but they use the h1 to h6 opening and closing tags.

  12. How many different levels of headings are there and what is the difference between them?
    • There are a range of 6 heading sizes. h1 to h6. The higher the number the smaller the heading font size becomes.

  13. What element should you use to make text bold and important?
    • The "b" opening and closing tags can accomplish this, but it is better for screen readers to use the "strong" tags instead.

  14. What element should you use to make text italicized to add emphasis to it?
    • Browsers will emphasize text that is wrapped in the "em" opening and closing tags.

  15. What relationship does an element have with any nested elements within it?
    • It creates a parent and child relationship between them. The nested elements are the children and the element they are nested within is the parent.

  16. What relationship do two elements have if they are at the same level of nesting?
    • Elements at the same level of nesting are considered to be siblings.

  17. How do you create HTML comments?
    • The comment tag is used to insert comments in the source code. Comments are not displayed in the browsers.

  18. What HTML tag is used to create an unordered list?
    • The "ul" opening and closing tag is used to create an unordered list.

  19. What HTML tag is used to create an ordered list?
    • The "ol" opening and closing tag is used to create an ordered list.

  20. What HTML tag is used to create list items within both unordered and ordered lists?
    • The "li" tag is used to create list items within either an ordered or an unordered list.

  21. What element is used to create a link?
    • The anchor element, with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.

  22. What is an attribute?
    • Attributes provide additional information about elements. Attributes are always specified in the start tag. Attributes usually come in name/value pairs like: name="value".

  23. What attribute tells links where to go?
    • The "href" attribute indicates the destination of the hyperlink. Without the href attribute, the a element won't work.

  24. What is the difference between an absolute and relative link?
    • An absolute URL contains all the information necessary to locate a resource. A relative URL locates a resource using an absolute URL as a starting point.

  25. Which element is used to display an image?
    • The "img" tag is used to display images in HTML.

  26. What two attributes do images always need to have?
    • The source location (URL) of the image file. alt : The alternate text. This is used to describe the image for someone who cannot see it because they are either using a screen reader or the image src is missing.

  27. How do you access a parent directory in a file path?
    • The parent folder will be the folder that is listed before the final slash in the file path. For example, if the file path is "C:\Users\username\Documents\myfile. xlsx," then the parent folder is "C:\Users\username\Documents."

  28. What are the four main image formats that you can use for images on the web?
    • JPEG (or JPG) - Joint Photographic Experts Group. PNG - Portable Network Graphics. GIF - Graphics Interchange Format. TIFF - Tagged Image File.
